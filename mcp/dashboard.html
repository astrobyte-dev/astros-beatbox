What is this?

It's TidalCycles — music as code. Each box (d1..d16) is one looping layer.

The knobs rewrite a layer's code live. The console (bottom) runs anything you type.

Pattern (in "quotes")

bd a sound · ~ rest · spaces split the bar · bd*4 4× · [a b] group · <a b> alternate per bar

Keys & effects

Space = all off/on · 1-9 = toggle one layer · c = cheat sheet · ? = this guide

🔇 All off mutes everything; each layer's M (or keys 1-9) brings it in/out one at a time. ■ Stop clears all.
